# 环境搭建

## 方法一：BN Online（推荐）
1. 打开 [BN Online](https://hhcl233.github.io/betternemo-online/)
2. 按 F12 打开 DevTools
3. 在 Console 中调试扩展，或在 Sources 中断点调试
4. 无需安装，浏览器即可直接测试

## 方法二：Android 设备仿真
在 PC 浏览器 DevTools 中：
1. 开启设备仿真（Toggle Device Toolbar）
2. 添加自定义设备，User Agent 设置为 `__TEST_ENV__`
3. 刷新页面即可进入测试环境

> `__TEST_ENV__` UA 会触发 BN 的测试模式，显示额外的调试信息。
