#### 前言

好的扩展，一定要对积木的 **样式**  进行改造

从使用者方面来说，积木文案太长，功能相似的积木太多，那么这个扩展 **一定 ** 难用

花一点时间改造样式，这会让你的控件**更受欢迎**

本文档规范了积木颜色，文案、功能合并、标签等

####   颜色

- 让使用者了解这块功能的**大概用处**

- 不要使用透明，亮色导致积木**看不清**

| 常量名 | 用途 |
| --- | --- |
| BLUE_3 | 事件积木 |
| ORANGE_2 | 处理数据，运算，字符串相关的功能 |
| ORANGE_1 | 列表相关的功能 |
| PURPLE_3 | 字典、音乐、视频相关的功能 |
| BLUE_6 | 功能 |
| ORANGE_4 | 函数相关的功能，一次执行多个指令 |
| PURPLE_1 | 设置数据的属性 |
| BLUE_1 | 请求数据，调用API，控制某些数据 |
