# 自定义扩展

## 准备

你应该有一个扩展文件夹或压缩包，如果是压缩包请把它解压。

## 安装方式

### BetterNemo Online
1. 打开 [BN Online](https://hhcl233.github.io/betternemo-online/)
2. 打开或新建一个作品
3. 通过扩展加载功能加载扩展的 URL

### Android 端
1. 使用 MT 管理器打开 BN apk → 查看
2. 进入 `/assets/webview/extensions/`
3. 将扩展 .js 文件放入该目录
4. 编辑 `_CONFIG.js` 注册扩展
5. 保存并重新安装

> 旧版安装教程见 [自定义扩展(旧版)](/扩展/自定义扩展(旧版))（260215.3 版本前适用）

## 获取扩展

- 在交流群中查找他人分享的扩展
- 参考 [扩展详解](/扩展/开发/扩展详解) 自行开发
