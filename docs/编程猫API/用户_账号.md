# 用户/账号
# 🔐登录

## [ 1 ]

> `api` `POST` `/tiger/accounts/login`

基础的账号密码登录。

### ⚙️请求体

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| ❗identity | 账号 | String |
| ❗pasword | 密码 | String |
| ❗pid | 登录地标识 | String |

### 📖响应内容

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| token | 登录token | String |
| cmid |  | String |
| user_info | 用户信息 | Object |
| expiration_time | 过期时间戳 | Number |

#### user_info

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| id | 训练师编号 | Number |
| nickname | 昵称 | String |
| avatar_url | 头像链接 | String |
| fullname | 真实姓名 | String |
| birthday | 生日时间戳 | Number |
| sex | 性别（0：女，1：男） | Number |
| qq | QQ号 | String |
| description | 简介 | String |
| grade | ❓年级 | Number |
| grade_desc | ❓年级 | String |

### ↪️响应示例

```json
{
  "token": "eyJ...",
  "cmid": "",
  "user_info": {
    "id": 114514,
    "nickname": "用户昵称",
    "avatar_url": "https://cdn-community.bcmcdn.com/47/community/xxx.png",
    "fullname": "全名",
    "birthday": 114514,
    "sex": 0,
    "qq": "114**********",
    "description": "​",
    "grade": 8,
    "grade_desc": "零年级"
  },
  "expiration_time": 114514
}
```

## [ 2 ]

> `api` `POST` `/tiger/v3/web/accounts/login`

Web端账号密码登录，返回Cookie。

### ⚙️请求体

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| ❗identity | 账号 | String |
| ❗pasword | 密码 | String |
| ❗pid | 登录地标识 | String |

### 📖响应内容

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| auth | token相关 | Object |
| user_info | 用户信息 | Object |

#### auth

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| token | 登录token | String |
| phone_number | 电话号码（前3后4） | String |
| email | 邮箱 | String |
| has_password | 是否设置密码 | Boolean |
| is_weak_password | 是否为弱密码 | Boolean |

#### user_info

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| id | 训练师编号 | Number |
| nickname | 昵称 | String |
| avatar_url | 头像链接 | String |
| fullname | 真实姓名 | String |
| birthday | 生日时间戳 | Number |
| sex | 性别（0：女，1：男） | Number |
| qq | QQ号 | String |
| description | 简介 | String |
| grade | ❓年级 | Number |
| grade_desc | ❓年级 | String |

### ↪️响应示例

```json
{
  "auth": {
    "token": "eyJ...",
    "phone_number": "114****9198",
    "email": "",
    "has_password": true,
    "is_weak_password": true
  },
  "user_info": {
    "id": 114514,
    "nickname": "用户昵称",
    "avatar_url": "https://cdn-community.bcmcdn.com/47/community/xxx.png",
    "fullname": "全名",
    "birthday": 114514,
    "sex": 0,
    "qq": "114**********",
    "description": "​",
    "grade": 8,
    "grade_desc": "零年级"
  }
}
```
