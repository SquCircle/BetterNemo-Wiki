# 📢声明

### 🏷️凡例

一些API会标注**BaseURL** 和**请求方式** 。在使用这些API时，请从下表获取对应**BaseURL** 标签的域名，并添加在API路径的前方。

对于未标注**BaseURL** 标签的API，会将域名和API路径一同提供。

| BaseURL标签 | 域名 |
| --- | --- |
| api | https://api.codemao.cn |

**请求参数/请求体** 中，以`❗`开头的项为**必填项** 。

**返回内容/响应内容** 中，列表留空的项为**未知项** ；以`❓`开头的项为**猜测项** 。
