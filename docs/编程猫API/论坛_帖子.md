# 🗂️论坛板块

## [ 1 ]

> `api` `GET` `/web/forums/boards/simples/all`

获取论坛所有板块的简讯。

### 📖返回内容

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| items | 板块信息 | Array |

#### items

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| id | 板块id | String |
| name | 板块名称 | String |
| icon_url | 板块图标链接 | String |
| is_hot | 是否为热门板块 | Boolean |
| has_popular |  | Boolean |
| has_selection |  | Boolean |
