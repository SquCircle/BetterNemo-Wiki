# 杂项
# ⏱️获取服务器时间戳

> `api` `GET` `/coconut/clouddb/currentTime`

获取当前时刻服务器的秒时间戳。

### 📖返回内容

| **KEY** | **VALUE** | **TYPE** |
| --- | --- | --- |
| code | http状态码 | Number |
| success | 业务是否成功 | Boolean |
| data | 时间戳 | Number |
| msg | 操作结果消息描述 | String |
| extData | 额外信息 |  |
| traceID | 请求追踪id | String |

### ↪️返回示例

```json
{
  "code": 200,
  "success": true,
  "data": 1777102806,
  "msg": "操作成功",
  "extData": null,
  "traceId": "53a6b7d5725b4d68ae2ae37aad0c6e92.46631.17771028063236415"
}
```
