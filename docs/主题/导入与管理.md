# 导入与管理

## 方法一：BN Online
[BetterNemo Online](/BN%20Online/README) 可通过 MDUI 颜色选择器实时调整编辑器主题色，无需手动替换文件。

## 方法二：Android 手动安装
1. 使用 [MT管理器](https://mt2.cn/) 打开 BetterNemo 的apk，选择查看

2. 进入路径 `/assets/webview/theme/`

3. 将主题文件夹放入该路径，并在 `_CONFIG.js` 中注册

4. 安装，进入 BetterNemo 检查安装情况
