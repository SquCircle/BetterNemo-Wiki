# 获取Webview版本
# 安卓端

## 打开开发者模式

以ColorOS 14.0为例：

设置→关于本机→版本信息→连续点击版本号，直到提示“你已处于开发者模式”。

## 查看Webview版本

设置→其他设置→开发者选项→WebView实现。

## 更新WebView

1. Google Play（某些系统显示为Play商店）搜索Android System Webview。

2. Apkpure/Apkmirror等第三方平台下载。

*请**留意安装包版本，小心误下载x86版。以上部分操作需要魔法上网，请自行解决。**国内部分安卓系统魔改了Webview，可能无法正常更新。*
